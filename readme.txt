= Features of WooCommerce Gift Card Lite =

* **Gift card products:**  Merchant can create gift card as an all-new product via WooCommerce.
* **Price variation:**  The user can create four types of price variation types like-default price,  user price,  price range & selected price.
* **Multiple Gift Card:**  Merchant can create the multiple numbers of gift cards on the basis of occasion for the anniversary, birthday or any festivity.
* **Gift card prefix:**  This allows to enter the name of the brand in front of the coupon code.
* **Gift card length:**  This can be set by the admin that whether the gift card coupon code would be lengthy or short. 
* **Gift card expiration:**  The recipient has to purchase the product prior to the expiry date so that he could enjoy the best of the services within the noted time.
* **Minimum spend:**  The merchant can fix the minimum limit on coupon money within those limit client can utilize the coupon money.
* **Maximum spend:**  The admin can fix the maximum limit on coupon money within those limit customer can utilize the money.
* **Gift card usage:**  This is set by the admin that about how many times you can use a gift card coupon code.

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don't need to leave your web browser. To do an automatic install of Woocommerce Gift Cards Lite, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type "Woocommerce Gift Cards Lite" and click Search Plugins. Once you've found our Woocommerce Gift Cards Lite plugin you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking "Install Now".

= Manual installation =

The manual installation method involves downloading our Woocommerce Gift Cards Lite and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).
